# by.svs1e — Handmade with Love 🍓

Custom website for **by.svs1e** — handmade bracelets, keychains & charms.

## Setup

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub, then import into [Vercel](https://vercel.com). Vercel auto-detects Vite and handles everything.

## Stack

- React + Vite
- Google Fonts (Playfair Display + DM Sans)
- No external UI libraries — custom styled
